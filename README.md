# NeetCode 150 🚀

Este repositorio contiene mis soluciones a la lista de problemas de [NeetCode 150](https://neetcode.io/roadmap). El objetivo principal es practicar algoritmos, estructuras de datos y mejorar mis habilidades de resolución de problemas de cara a entrevistas técnicas.

## 📁 Estructura del Proyecto

Las soluciones están divididas por lenguajes de programación, para mantener los entornos separados y limpios:

*   **🐍 `Python/`**: Resoluciones utilizando Python, priorizando código limpio (siguiendo PEP 8) y eficiencia.
*   **📘 `TypeScript/`**: Resoluciones utilizando TypeScript, aprovechando el tipado estático y las convenciones del ecosistema moderno de JavaScript.

Dentro de cada lenguaje, los ejercicios están organizados según los temas del roadmap oficial (ej. `arrays-and-hashing`, `two-pointers`, `sliding-window`, etc.).

## 🎯 Meta

Dominar los patrones más comunes de algoritmos y reducir el tiempo de resolución para problemas óptimos.
